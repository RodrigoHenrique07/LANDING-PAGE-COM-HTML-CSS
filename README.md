# LANDING-PAGE-BALLE-BOT

Projeto desenvolvido junto com a rafaellaBallerine, onde o prinipal intuíto foi melhorar o entendimento sobre o html semântico, dividando o projeto em header, 
main e footer além de usar a tag nav para identifcar a área de navegação, bem como usar o CSS para criar estilos e posicionar os alementos com o flex-box. Além disso, também foi usado a responsividade

# [Tecnologias utilizadas](#tecnologias)
* <p> HTML como foco no html semântico </p>
* <p> CSS com foco no flex-box </p>
